# FoodDrop — Drop Platform for Food Creators

A CRM and distribution platform for small food creators to manage "drops" — scheduled food releases where customers can order and pick up items.

## Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```

### 3. Deploy to Vercel
Push this repo to GitHub, then connect it to [Vercel](https://vercel.com) for automatic deployment.

## Tech Stack
- **Frontend:** React + Vite
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Vercel (recommended)

## Database Setup
Run the `supabase-schema.sql` file in your Supabase project's SQL Editor to create the required tables.
